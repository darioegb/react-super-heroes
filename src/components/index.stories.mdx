# Shared Components

They are shared react components based on material ui.
They are grouped according to their functionality.
Some components use internationalization with [react-i18next](https://react.i18next.com/)

## Form

They are the components related to forms.
For manage form fields is used [react-hook-form](https://react-hook-form.com/)
Also for storage files is using [firebase-storage](https://firebase.google.com/docs/storage)

## Grid

They are the components related to the data grid.
